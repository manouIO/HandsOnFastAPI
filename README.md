# HandsOnML-Linux-PyTorch-SQL
This repository is dedicated to learning and practical exercises on Linux, PyTorch, and SQL.

Content Overview:
Linux: Basics of Linux command-line usage, Bash scripting, shell commands, and automation techniques.

PyTorch: Hands-on deep learning classification projects, neural network implementations, and torchmetrics usage.

SQL: Database design, querying, schema management, and integrating SQL with Python.
